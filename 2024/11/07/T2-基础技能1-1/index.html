Linux 背景简介

Linux 的应用领域

相比 Windows 等操作系统（OS），Linux 的轻量化特性广泛应用于许多系统和设备。

Linux 的各种版本

Linux 是一个统称，包含基于 UNIX 的多个操作系统。由于 Linux 的开源特性，衍生出许多适用于不同需求的版本。

运行命令

Ubuntu 等轻量操作系统的优点是轻量化，这意味着通常没有图形用户界面（GUI）或桌面环境。大部分操作都通过终端纯文本界面完成

命令 描述
echo 输出指定文本
whoami 查看当前登录的用户

echo 命令

使用 echo 命令在终端输出内容

whoami 命令

使用 whoami 命令查找当前登录用户


操作文件系统

命令 全称
ls 列出目录内容（listing）
cd 更改目录（change directory）
cat 拼接并显示内容（concatenate）
pwd 显示当前目录（print working directory）

ls 命令

使用 ls 命令列出当前目录下的所有内容

可以直接使用 ls 和目录名称查看某个目录内容，而不必进入该目录

cd 命令

使用 cd 命令切换到其他目录

cat 命令

使用 cat 命令查看文件的内容

可以用 cat 命令直接输出指定路径文件的内容，而不必进入该目录

pwd 命令

使用pwd 命令找到完整路径


搜索文件

find 命令

使用 find 命令进行简单和复杂搜索

如果知道文件名，可以使用以下命令来查找

如果不记得文件名，可使用通配符 * 查找所有扩展名的文件

grep 命令

使用 grep 命令用于在文件内容中搜索特定的值


Shell 操作符简介

操作符 描述
& 在后台运行命令
&& 串联多个命令，仅在前一个命令成功时执行下一个命令
> 输出重定向，将命令输出覆盖写入文件
>> 输出重定向，将命令输出追加到文件末尾

& 操作符

使用 & 操作符在后台执行命令，从而继续执行其他操作

&& 操作符

使用 & 操作符串联多个命令，只有成功运行第一个命令时，第二个命令才会运行

> 操作符

使用 > 操作符输出重定向，将命令输出重定向至文件。如果文件已存在，则会覆盖文件内容。

>> 操作符

使用 >> 操作符输出重定向符，与 > 类似，但会将内容追加到文件末尾，而不覆盖现有内容。
