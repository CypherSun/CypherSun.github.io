Windows 更新

让我们从Windows 更新开始。

Windows 更新是 Microsoft 提供的一项服务，为 Windows 操作系统和其他 Microsoft 产品（例如 Microsoft Defender）提供安全更新、功能增强和补丁。

更新通常在每个月的第二个星期二发布。这一天称为 补丁星期二。这并不一定意味着关键更新/补丁必须等到下一个补丁星期二发布。如果更新很紧急，那么微软会通过 Windows 更新服务将更新推送到 Windows 设备。

请参阅以下链接查看此处的Microsoft 安全更新 指南。

Windows 更新位于“设置”中。请参见下文。

提示：访问 Windows 更新的另一种方法是从运行对话框或 CMD 运行命令 control /name Microsoft.WindowsUpdate。

在附加的VM中，有几件事需要强调。

Windows 更新设置是“受管理的”。（通常，家庭用户不会看到此类消息）
虚拟机没有可用的更新。（附加的虚拟机无法访问 Internet，因此无法与 Microsoft 通信以获取新更新）

多年来，Windows 用户已经习惯于将 Windows 更新推迟到以后的日期或根本不安装更新。导致此操作的原因有很多，其中之一是 Windows 更新后通常需要重新启动。

微软在 Windows 10 中特别解决了这个问题。这些更新不能再被忽略或推到一边直到被遗忘。Windows 更新只能推迟，但最终更新会发生，你的电脑会重新启动。微软提供这些更新是为了确保设备的安全。

防火墙和网络保护

什么是防火墙？

根据微软的说法，“流量通过我们所谓的端口流入和流出设备。防火墙控制着哪些东西可以通过这些端口，更重要的是哪些东西不能通过。你可以把它想象成站在门口的保安，检查所有试图进出的人的身份证”。

下图将反映您导航到防火墙和网络保护时看到的内容。

注意：每个网络可能具有不同的状态图标。

这三个（域，私有，和公共）之间有什么区别？

根据微软的说法，“ Windows 防火墙提供三种防火墙配置文件：域、私人和公共”。

域- 域配置文件适用于主机系统可以向域控制器进行身份验证的网络。
私人- 私人配置文件是用户分配的配置文件，用于指定私人或家庭网络。
公共-默认配置文件是公共配置文件，用于指定公共网络，例如咖啡店、机场和其他地点的 Wi-Fi 热点。
如果您单击任何防火墙配置文件，将出现另一个屏幕，其中有两个选项：打开/关闭防火墙并阻止所有传入连接。

警告：除非您对自己所做的事情100%有信心，否则建议您保持 Windows Defender防火墙处于启用状态。

安全处理器

以下是安全处理器的详细信息。

什么是可信平台模块( TPM )？

根据微软的说法，“可信平台模块 (TPM) 技术旨在提供基于硬件的安全相关功能。TPM 芯片是一种安全的加密处理器，旨在执行加密操作。该芯片包含多种物理安全机制，使其具有防篡改功能，恶意软件无法篡改 TPM 的安全功能”。

BitLocker

什么是BitLocker？

根据微软的说法，“ BitLocker 驱动器加密是一种与操作系统集成的数据保护功能，可解决因计算机丢失、被盗或不当退役而导致的数据盗窃或泄露威胁”。

在安装了 TPM 的设备上，BitLocker 提供最佳保护。

根据 Microsoft 的说法，“ BitLocker 与可信平台模块 (TPM) 1.2 版或更高版本一起使用时可提供最强的保护。TPM 是计算机制造商在许多较新的计算机中安装的硬件组件。它与 BitLocker 配合使用，帮助保护用户数据并确保计算机在系统脱机时不会被篡改”。

请参阅 Microsoft 官方文档以了解有关 BitLocker 的更多信息。

注意：附加的VM中不包含 BitLocker 功能。

卷影复制服务

根据Microsoft的说法， 卷影复制服务 (VSS) 协调所需的操作来创建要备份的数据的一致卷影副本（也称为快照或时间点副本）。

卷影副本存储在每个启用了保护的驱动器上的系统卷信息文件夹中。

如果启用了 VSS（打开了系统保护），您可以在高级系统设置中执行以下任务。

创建还原点
执行系统还原
配置恢复设置
删除还原点
从安全角度来看， 恶意软件编写者知道 Windows 的这一功能，并在其恶意软件中编写代码来查找这些文件并将其删除。这样做会使用户无法从勒索软件攻击中恢复，除非用户有离线/异地备份。
