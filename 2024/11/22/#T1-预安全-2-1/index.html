什么是网络？

网络就是将事物连接起来。例如，朋友圈是因为相似的兴趣、爱好、技能和类型而联系在一起。

网络存在于各行各业：
城市的公共交通系统
国家电网等基础设施
与邻居见面并问候
用于寄送信件和包裹的邮政系统

但更具体地说，在计算领域，网络是同一个概念，只是分散到了技术设备上。以手机为例；拥有它的原因是为了能访问事物。

在计算领域，网络可以由 2 台设备到数十亿台设备组成。这些设备包括笔记本电脑、手机、安全摄像头、交通信号灯等一切设备。

网络已融入日常生活。无论是收集天气数据、向家庭输送电力，还是确定谁在道路上拥有通行权。由于网络已深深植根于现代生活，因此网络是网络安全中必须掌握的一个基本概念。

网络有各种形状和大小。

什么是互联网？

互联网是一个由许多小网络组成的巨型网络。现在让假设 Alice 结识了一些新朋友，他们分别叫 Zayn 和 Toby，她想把他们介绍给 Bob 和 Jim。问题是 Alice 是唯一一个与 Zayn 和 Toby 说同一种语言的人。所以 Alice 必须成为信使。由于 Alice 会说两种语言，所以他们可以通过 Alice 互相交流——形成一个新的网络。

互联网的第一次迭代是在 20 世纪 60 年代末的 ARPANET 项目中。该项目由美国国防部资助，是第一个有记录的网络。然而，直到 1989 年，所知的互联网才由 Tim Berners-Lee 通过创建万维网 (WWW) 发明。直到此时，互联网才开始像今天一样被用作存储和共享信息的存储库。

如前所述，互联网是由许多连接在一起的小型网络组成的。这些小型网络称为专用网络，连接这些小型网络的网络称为公共网络 - 或互联网。因此，总结一下，网络可以分为以下两种类型：
专用网络
公共网络

设备将使用一组标签在网络上识别自己。

识别网络上的设备

为了沟通和维持秩序，设备在网络上必须既具有识别性又可识别。如果到头来都不知道在和谁说话，那还有什么用呢？

网络上的设备与人类非常相似，有两种识别方式：
名字
指纹

现在可以更改姓名，但无法更改指纹。每个人都有一组单独的指纹，这意味着即使他们更改了姓名，其背后仍然有身份。设备也有同样的功能：两种识别方式，其中一种是可渗透的。它们是：
IP 地址
媒体访问控制 (MAC) 地址 - 可以将其视为类似于序列号。

IP 地址

简而言之，IP 地址或 Internet 协议地址可在一段时间内用作识别网络上主机的一种方式，然后该 IP 地址可与另一台设备关联，而无需更改 IP 地址。

IP地址是一组分为四个八位字节的数字。每个八位字节的值将汇总为网络上设备的 IP 地址。该数字是通过称为 IP 寻址和子网划分的技术计算出来的。IP 地址可以在不同设备之间更改，但不能在同一网络中同时处于活动状态多次。

IP 地址遵循一组称为协议的标准。这些协议是网络的骨干，迫使许多设备使用同一种语言进行通信。但是，应该记住，设备可以位于私有和公共网络上。根据它们所在的位置，将确定它们拥有哪种类型的 IP 地址：公共或私有 IP 地址。

公共地址用于在互联网上识别设备，而私有地址用于在其他设备中识别设备。以下面的表格为例。这里在私有网络上有两个设备：

设备名称 IP 地址 IP 地址类型
桌面式-KJE57FD 192.168.1.77 私有
桌面式-KJE57FD 86.157.52.21 公共
CMNatic-PC 192.168.1.74 私有
CMNatic-PC 86.157.52.21 公共

这两个设备将能够使用其私有 IP 地址相互通信。但是，从这两个设备发送到互联网的任何数据都将由相同的公共 IP 地址标识。公共 IP 地址由互联网服务提供商 (ISP)提供，每月账单收费

随着越来越多的设备联网，获取尚未使用的公共地址变得越来越困难。互联网上将有大约 500 亿台设备联网。输入 IP 地址版本。到目前为止，只讨论了互联网协议寻址方案的一个版本，即 IPv4，它使用42.9 亿个 IP 地址的编号系统——所以可以看到为什么会有如此短缺

IPv6 是互联网协议寻址方案的新版本，可帮助解决此问题。虽然它看起来更令人生畏，但它具有一些优势：
支持超过 340 万亿个 IP 地址，解决 IPv4 面临的问题
采用新方法，效率更高

MAC 地址

网络上的所有设备都具有物理网络接口，即设备主板上的微芯片板。此网络接口在出厂时被分配了一个唯一的地址，称为媒体访问控制 (MAC) 地址。MAC 地址是一个十二个字符的十六进制数，即一种用于计算的十六进制数字系统，分成两部分并用冒号分隔。这些冒号被视为分隔符。例如，a4:c3:f0:85:ac:2d。前六个字符代表制造网络接口的公司，后六个字符是一个唯一的数字。

然而，MAC 地址的一个有趣之处在于，它们可以被伪造或“欺骗”，这个过程被称为欺骗。当联网设备假装使用其 MAC 地址识别为另一台设备时，就会发生这种欺骗。当发生这种情况时，它通常会破坏实施不佳的安全设计，这些设计假设网络上通信的设备是值得信赖的。例如，防火墙配置为允许任何往返于管理员 MAC 地址的通信。如果设备假装或欺骗此 MAC 地址，防火墙现在会认为它正在接收来自管理员的通信，但事实并非如此。

咖啡馆、咖啡店和酒店等场所在使用访客或公共 Wi-Fi 时经常使用 MAC 地址控制。如果愿意按设备付费，此配置可以提供更好的服务，即以一定价格获得更快的连接。

Ping（ICMP）

Ping 是可以使用的最基本的网络工具之一。Ping 使用Internet 控制消息协议 (ICMP) 数据包来确定设备之间连接的性能，例如，连接是否存在或是否可靠。

ICMP 数据包在设备之间传输所需的时间可通过 ping 来测量。此测量使用 ICMP 的回应数据包以及来自目标设备的 ICMP 回应回复来完成。

例如家庭网络或网站等资源，可以对网络上的设备执行 Ping。此工具易于使用，并安装在 Linux 和 Windows 等操作系统 (OS) 上。执行简单 Ping 的语法是 ping IP 或者网站 URL。
