变量和数据类型

变量允许在程序中存储和更新数据，通过变量名可访问和管理存储的数据

存储在变量中的数据的类型包括:

字符串（String）：用于存储字符
整数（Integer）：表示整数值
浮点数（Float）：表示小数或分数
布尔值（Boolean）：通常用于条件判断，存储布尔值 True 或 False
列表（List）：允许按位置存储多个值，用于存储不同数据类型的集合

if语句

使用 if 语句可让程序根据条件判断来选择不同的执行路径

if 关键字表示语句开始，后面跟随条件
冒号 : 表示 if 语句结束
if 语句检查条件是否为真，如果为真，则执行相应的代码块，否则可以执行 else 或 elif 分支中的其他代码块。
任何在冒号后缩进的内容都被视为 if 语句的一部分，程序将执行这些内容

循环

循环可以让程序多次重复执行操作，主要有两种类型的循环：for 循环和 while 循环

While 循环

while 循环基于条件判断执行，用于在条件满足时执行，直到条件不满足为止
当条件始终为 True 时，while 循环会无限次运行
当条件变为 False 时，循环会停止

while 语句指定了循环的起始位置
每次循环，程序从循环顶部开始
接着程序执行循环中的下一行
没有更多代码执行时，程序回到循环顶部，重复上述过程

For 循环

for 循环用于遍历序列或根据固定次数执行循环

循环遍历每个元素
当循环遍历完列表中的所有元素时，程序停止循环

使用 range 函数生成一系列数值，然后在 for 循环中遍历数值


函数介绍

函数是一段代码块，可在程序的不同位置被调用，用于执行特定的任务。函数可接收输入参数，处理数据，并返回结果

def 关键字表示函数的开始。函数名和参数由程序员定义
在函数名后面有一对括号 ()，其中包含可以传递给函数的数据
冒号 : 表示函数头的结束
冒号后缩进的内容都被视为函数的一部分
函数会返回一个结果

文件

常用的文件操作模式包括读取、写入和追加等，文件操作通过内置的 open() 函数完成

可用 readlines() 方法并循环遍历文件中的每一行，如果文件位于其他位置，需要提供文件的完整路径

要写入新文件，使用 “w”参数表示“写入”（write）
要追加内容，需先打开文件，并在 open() 函数中使用 “a” 参数表示“追加”（append）

写入文件后使用 close() 函数关闭文件，防止在程序中进一步对该文件进行写入操作

导入

库是一组包含函数、类和变量的文件。通过将库导入到程序中，可使用库中已定义的函数和功能

使用 import 关键字来导入库，通过引用库名调用其函数

渗透测试人员的常用库：

Request - 简单的 HTTP 库。https://github.com/psf/requests
Scapy - 用于发送、嗅探、解析和伪造网络数据包。https://github.com/secdev/scapy
Pwntools - 一个用于 CTF 和漏洞开发的库。https://github.com/Gallopsled/pwntools

许多库已内置于 Python 编程语言中，对于未预装的第三方库，可使用 pip 工具进行安装，pip 是 Python 的包管理器
