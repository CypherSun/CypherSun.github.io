Linux 背景知识简介

与 Windows 等其他操作系统（OS）相比，各有优缺点。Linux 是多个类 UNIX 操作系统的统称。由于 Linux 是开源的，因此有许多不同的 Linux 发行版

基础命令

命令 描述
echo 输出提供的任何文本
whoami 查看当前登录的用户身份

使用 echo 命令，输出文本

使用 echo 命令输出字符串

使用 whoami 命令，查看当前登录的用户身份

使用 whoami 命令查看当前登录的用户身份


与文件系统的交互命令

命令 描述
ls 列出当前目录中的文件（listing）
cd 更改当前目录（change directory）
cat 输出文件内容（concatenate）
pwd 显示当前工作目录的完整路径（print working directory）

使用 ls 命令

使用 ls 命令来列出当前目录的文件
使用 ls 命令和目录名称不进入该目录列出目录的文件

使用 cd 命令

使用 cd 命令更改当前目录

使用 cat 命令

使用 cat 命令来输出文件内容
使用 cat 命令和目录路径不进入该目录输出目录中文件内容

使用 pwd 命令

使用 pwd 命令显示当前工作目录的完整路径


查找文件命令

命令 描述
find 在当前目录所有子目录中搜索文件
grep 搜索整个文件中包含特定值的行（Global Regular Expression Print）

使用 find 命令

如果记得文件名，使用 find -name 命令在当前目录所有子目录中搜索文件
如果不记得文件名，可搜索所有具有特定扩展名的文件，使用 find -name 命令和通配符（*）来搜索所有以特定扩展名结尾的文件

使用 grep 命令

使用 grep 命令搜索整个文件内容来查找包含特定值的行


字符串内注意需要完全符合大小写

Shell 操作符

符号/操作符 描述
& 执行命令并在后台运行
&& 将多个命令结合在一行中执行（仅前一命令成功执行才能继续）
> 一个重定向符，可以将某个命令的输出重定向到其他地方（覆盖原有内容）
>> 功能类似于 >，但会将输出追加而不是替换（不会覆盖原有内容）

使用 & 操作符

使用 & 操作符执行命令并在后台运行

使用 && 操作符

使用 && 操作符创建命令链，只有第一个命令成功执行后，第二个命令才会执行

使用 > 操作符

使用 > 操作符输出重定向，可以将命令的输出保存到指定的文件或其他地方

如果文件已存在，内容会被替换
如果文件不存在，会创建该文件，并将输出写入其中

使用 >> 操作符

使用 >> 操作符输出重定向，并会将内容追加到文件末尾，而不会覆盖文件原有的内容
