什么是网络？

网络就是连接在一起的事物。在计算机领域，网络的概念相同，连接的是各种技术设备，网络可以由2台设备到数十亿台设备组成


什么是互联网？

互联网是一个巨大的网络，由许许多多的小型网络组成。

互联网的首次雏形出现在1960年代末的ARPANET项目中。该项目由美国国防部资助，是首个实际应用的网络。直到1989年，由Tim Berners-Lee发明了万维网（World Wide Web，WWW），互联网开始被用作存储和共享信息的存储库。

互联网是由许多小型网络相互连接而成的。这些小型网络称为私有网络，而连接这些小网络的网络称为公共网络，也就是互联网。

网络可以分为两种类型：
私有网络
公共网络


识别网络中的设备

网络中的设备和人类非常相似，有两种识别方式：
名字
指纹

设备有两种身份识别方式，其中一种是可更改的：

IP地址
媒体访问控制（MAC）地址设备的序列号

IP地址

IP 地址 (Internet Protocol) 可在一段时间内用作识别网络上主机的一种方式，该 IP 地址可与另一台设备关联，而无需更改 IP 地址

IP 地址是一组分为四个八位字节的数字，每个八位字节的值将汇总为网络上设备的 IP 地址。该数字是通过称为 IP 寻址和子网划分的技术生成，IP 地址可以在不同设备之间更改，但不能在同一网络中同时处于活动状态多次

IP 地址遵循一组协议的标准。这些协议是网络的基础，使得多个设备可交流。设备可处于私有网络或公共网络中，而其所处的网络类型决定了它拥有的 IP 地址类型：公共 IP 地址或私有 IP 地址

公共地址用于在互联网上识别设备，而私有地址则用于在设备间的相互识别。

这两个设备将能够使用其私有 IP 地址相互通信。任何从这些设备发送到互联网的数据都会通过相同的公共IP地址进行标识。公共 IP 地址是由互联网服务提供商（ISP）分配的，按月收费。

IPv4 是互联网协议，它使用2^32个 IP 地址（42.9亿）

IPv6 是互联网协议寻址方案的新版本，它使用2^128个IP地址（超过340万亿个），解决了 IPv4 IP 地址不够分配的问题

MAC地址

网络上的所有设备都具有物理网络接口，设备主板上的微芯片板。该网络接口在出厂时被分配了一个唯一的地址，称为 MAC（媒体访问控制）地址。MAC 地址是一个十二字符的十六进制数字，每两个字符一组，用冒号分隔。前六个字符代表制造该网络接口的公司，后六个字符是唯一编号。

MAC 地址是可伪造或欺骗的，这一过程称为伪装（spoofed）。当网络设备使用其他设备的 MAC 地址假装自己是另一个设备时，就会发生伪装。

咖啡馆、酒店等场所的访客或公共 Wi-Fi 使用 MAC 地址控制。这种配置可以提供更好的服务


Ping（ICMP）

Ping 是可用的最基础的网络工具之一。Ping 使用互联网控制消息协议（Internet Control Message Protocol，ICMP）数据包来确定设备之间的连接性能

Ping 测量 ICMP 数据包在设备之间的往返时间来评估连接的表现

Ping 可用于测试网络中的设备或网站等资源。Ping 在 Linux 和 Windows 等操作系统中预装。
