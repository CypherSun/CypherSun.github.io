什么是网络？

网络本质上是事物之间的连接关系。例如，因共同的兴趣、爱好或技能而相互连接的朋友圈，这也构成了一种网络。

网络无处不在，各个领域都能看到它的影子：

城市公共交通系统
国家电网等基础设施
与朋友的日常互动
快递的邮政系统

在计算机领域，网络的概念相似，只不过对象是技术设备。以手机为例，它能连接外界、获取信息。这些设备之间的通信有特定的规则和协议，网络技术就在于研究这些连接如何实现、通信如何进行，并且网络还有不同的形态和规模。

在计算机网络中，网络可以简单到仅由两台设备组成，也可以复杂到包含数十亿台设备。这些设备包括笔记本电脑、手机、监控摄像头、交通信号灯，或其他设备。


什么是互联网？

互联网是一个庞大的网络系统，由无数小型网络组成。

互联网的最初雏形可以追溯到 20 世纪 60 年代后期的 ARPANET 项目。该项目由美国国防部资助，是第一个实际运行的计算机网络。直到 1989 年蒂姆·伯纳斯-李发明了万维网（WWW），互联网才逐渐演变成今天。从那时起，互联网成为一个用于存储和共享信息的巨大存储库。

可以将朋友圈类比为小型网络，而互联网则是这些小型网络的集合体。互联网由许多私有网络构成，这些私有网络通过公共网络相互连接，形成一个全球性网络。

网络可以分为两种类型：
私有网络
公共网络

设备通过一组独特的标识在网络中识别自己。

识别网络中的设备

在网络上进行通信并保持秩序，设备必须既能识别他人，也能被他人识别。否则，如果无法确认交流对象，通信将变得毫无意义。

设备在识别上类似于人类，有两种识别方式：名字和指纹。虽然名字可以通过法律程序更改，但每个人的指纹却是独一无二且不可更改的，即使名字变了，身份仍任可以通过指纹确认。设备也有类似的识别方式：IP 地址和媒体访问控制（MAC）地址

IP 地址

IP 地址是互联网协议地址，用于在网络上识别设备，并可以在一定时间内关联到特定设备。它由四组数字组成，每组数字称为一个“八位组”，最终组合成一个设备的独特 IP 地址。地址分配遵循 IP 地址分配和子网划分的技术，确保在同一网络中每台设备拥有唯一的 IP 地址，防止地址冲突。

IP 地址分为公共 IP 地址和私有 IP 地址，具体取决于设备所在的网络类型。公共 IP 地址用于互联网上的识别，而私有 IP 地址通常在局域网内使用。分配这些地址并使用一套协议作为标准规范，协议是网络的核心，使得不同设备能够以统一的语言进行通信。MAC 地址则是设备的物理地址，通常不可更改，可以将其理解为设备的指纹或序列号。与 IP 地址不同，MAC 地址通常不变且独一无二，用于在同一网络的设备之间建立稳定的物理连接。

公共 IP 地址与私有 IP 地址

公共 IP 地址用于在互联网上识别设备，而私有 IP 地址用于在局域网内识别设备。例如两个设备能够在私有网络中各自拥有私有 IP 地址，同时并共享一个公共 IP 地址。这两个设备可以使用各自的私有 IP 地址在局域网内相互通信。然而，当它们中的任意一个设备向互联网发送数据时，都会通过同一个公共 IP 地址识别。这个公共 IP 地址是由互联网服务提供商（ISP）分配的，通常按月收费。

全球已有约 500 亿个设备连接到互联网，分配一个独特的公共 IP 地址变得越来越困难。因此不同版本的 IP 地址方案被提出以解决这一问题。目前常用的 IP 地址版本是 IPv4，它使用约 42.9 亿个地址。这一限制使得 IPv4 地址资源短缺。

IPv6 是一种新的互联网协议地址方案，旨在缓解 IPv4 地址分配的问题。IPv6 使用超过 340 万亿个，极大地增加了可用的 IP 地址数量。此外 IPv6 还引入了更高效的技术，使得数据传输和路由更优化。

MAC 地址

每个网络设备都具有一个物理网络接口，该接口包含一个出厂时分配的唯一地址，称为 MAC 地址。MAC 地址由 12 个十六进制字符组成，通常每两位一组，用冒号分隔，例如：a4:c3:f0:85:ac:2d。前六位代表制造商，后六位为设备的唯一序列号。

MAC 地址的一个特点是它可以被伪造，这种技术称为 MAC 地址欺骗。MAC 地址欺骗是指某设备通过模仿另一个设备的 MAC 地址来伪装身份。这种情况可能会破坏某些设计不完善的安全机制，例如，如果防火墙允许特定 MAC 地址的通信，攻击者可以通过 MAC 地址欺骗绕过该防火墙规则。

在一些公共场所，如咖啡馆、酒店等，Wi-Fi 网络通常使用 MAC 地址来管理访客接入，通过 MAC 地址识别和限制设备，以便控制网络访问或提供特定的连接服务。


Ping（ICMP）

Ping 是网络中的基本工具，用于检测设备之间的连接情况。它使用 ICMP（互联网控制消息协议）数据包来测试连接是否存在以及连接质量是否可靠。

Ping 的工作原理是通过测量 ICMP 数据包在设备之间的往返时间。Ping 发送一个 ICMP 的回显请求数据包给目标设备，目标设备收到后会回复一个回显回复数据包。通过这种往返过程，Ping 会测量从发送到接收的延迟时间。

Ping 可以对网络中的任何设备，如家庭网络中的电脑或互联网上的网站资源执行 Ping 操作。Ping 命令预装在 Linux 和 Windows 等操作系统中。基本语法是 ping IP地址 或 ping 网站 URL。
